Привіт. Це моя тестова сторінка
Мене звати Ростислав Хвостенко, я живу в Миколаївці Сумського району.

З 2005 по 2016 рік навчався в Миколаївській ЗОШ І–ІІІ ступенів, де активно цікавився інформатикою, брав участь у районних та обласних олімпіадах і неодноразово займав призові місця.

Після школи я вступив до машинобудівного коледжу СумДУ, а згодом продовжив навчання у Сумському державному університеті за спеціальністю «Матеріалознавство». В університеті отримав хорошу технічну базу, яка навчила мене системно мислити й аналізувати інформацію. Згодом я зрозумів, що хочу пов’язати своє життя з програмуванням. Для цього я вступив до Комп’ютерної Академії IT Step, де вивчав Python, алгоритми та структури даних. Це дало мені розуміння логіки програмування та заклало основу для подальшого розвитку.

Загалом я вивчав:

  • Python
  • JavaScript
  • React
  • HTML
  • CSS

Досягнення

  1. Призер районних та обласних олімпіад з інформатики.

Зараз я зосередився на веб-розробці. Вчуся створювати сучасні сайти, працюю з версткою та поступово освоюю інструменти frontend-розробника. Моя мета — розвиватися у сфері ІТ, удосконалювати свої навички та реалізовувати цікаві проєкти.
